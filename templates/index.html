Phone Number and email extractor


Instructions:

Copy the text that has the phone numbers and the emails that you would like to extract and paste them in the box.
App is designed around Tunisian phone numbers (+216) but it can extract any other country's 8 digit Phone number without the area code.
Note: in the current version, if you give it a number longer than 8 digits, it will give you the first 8 digits. (example: 123456789999 will return 12345678.) If the number can be cut in chunks of 8 numbers, it will do so. (Example: 123456781234567812345678 will return 12345678 three times)

App by amin HIDRI.
